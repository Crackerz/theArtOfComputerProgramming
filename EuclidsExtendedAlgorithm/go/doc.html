Package euclidext contains a simple implementation of Euclid's Extended Algorithm for finding a functions greatest common denominator. It is capable of solving the equation A*M+B*N=D for any input M and N where D is the greatest common denominator.

func EuclidExtended

func EuclidExtended(m, n int) (a, b, d int, e error)

Solves the equation A*M+B*N=D for any input M and N where D is the greatest common denominator. Returns A, B, D, and an error in the event of invalid input. If an error is encountered A, B, and D will be 0.
